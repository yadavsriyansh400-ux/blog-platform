# 🚀 Blog Platform (Full Stack)

A full-stack blogging platform where users can:
- Register & login securely
- Create, edit, and delete posts
- Upload images (Cloudinary)
- Comment on posts
- View user profiles

---

## 🛠️ Tech Stack

### Frontend
- React (Vite)
- Axios
- React Router

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication

### Other
- Cloudinary (Image Upload)

---

## ✨ Features

- 🔐 Authentication (JWT-based)
- 📝 Create/Delete Posts
- 🖼️ Image Upload Support
- 💬 Comment System
- 👤 Profile Page
- 🎨 Custom Gothic UI Theme
